## Setup
1. Virtuelle Umgebung aufsetzen
```python -m venv ./.venv```
2. Virtuelle Umgebung aktivieren
MacOS: ```source ./.venv/bin/activate```
Windows: ```./.venv/bin/Activate.ps1```
3. Packages installieren
```pip install -r requirements.txt```
4. App starten und loslegen
```python app.py```
